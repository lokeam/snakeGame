Hello! I'm Ahnming Loke and I'm a Front End Engineer.

This was my first attempt at making an HTML5 canvas game.

The challenge in this project was to see how much could be done within three hours time. Within about 180 minutes, I was able to figure out how to code a title screen and a workable prototype of the classic Snake video game that first appeared in arcades around 1976.

The application is coded in vanilla Object-Orientated javascript using EcmaScript 6 syntax to help with readability for the single class.

The design aesthetic for both this page and the game is inspired from the 8-bit area of NES home (ファミリーコン) console videogames.
